# Flask Template

A community standard Flask Template to use for initializing your Flask project. 

## Offering:

* Flask Project Folder Structure
* main.py
* app directory
* config directory
* routes directory
* utils directory
_* _init__.py in each directory for your modules

* .gitignore file
